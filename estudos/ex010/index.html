Formatações de texto

Temos a formatação em negrito. Pode ser usado com "b" quanto à forma

Temos a formatação em itálico. Pode ser usando com "i" quanto à forma

temos a formatação em marcação e temos 3 maneiras de mudarmos a cor dessa marcação, são elas:

  • Podemos mudar o estilo diretamente na tag.
  • Podemos mudar ela no documento no cabeçalho
  • Podemos mudar ela por links externos

Temos a formatação small para deixar textos pequenos

temos a formatação big para deixar textos grandes (Se tornou obsoleta)

Temos a formatação del para demonstrar que foi excluido o texto

temos a formatação ins para demonstrar que o texto foi inserido, onde pode utilizar o "U" quanto à forma

Temos a formatação sup para demonstrar textos Sobrescritos, ex.: x20+2 = a

Temos a formatação sub para demonstrar textos subscritos, ex.: H2O

Temos a formatação code para deixar uma fonte mono espaçada

Temos a formatação "Q" para demonstrar que estamos fazendo uma citação, podemos utilizar as asapas quanto à forma

Temos a formatação blockquote que serve para uma citação mais completa, ex.:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Est, doloribus quia nam itaque quos quasi, sapiente pariatur, ex excepturi commodi minus placeat esse? Quidem ab modi accusantium explicabo voluptatem esse.

Temos a formatação abbr para demonstrar as abreviações

Temos as listas ordenadas e não ordenadas.

Ordenadas:

  1. numeral
  1. Letras Maiúsculas
  1. Letras minúsculas
  1. Numeros romanos maiúsculos
  1. números romanos minúsculos
não ordenadas
  • Disco
  • Circulo
  • Quadrado
